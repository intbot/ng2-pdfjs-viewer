Embed style
Chromeless (pages only)
Install

npm i ng2-pdfjs-viewer

Steps
  1. Import PdfJsViewerModule (or the standalone component).
  2. Add <ng2-pdfjs-viewer> to a sized container.
  3. Bind [pdfSrc] to a URL, Blob, or byte array.

The viewer is 100% width/height of its parent — give the parent a height (e.g. height: 80vh).
